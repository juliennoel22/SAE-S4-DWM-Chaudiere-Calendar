SAE Atelier de développement d’application web
Semestre 4
L'objectif de la SAE est de développer une application web et mobile se composant
• d’un backend gérant des données et proposant des services accessibles pour certains via une
interface HTML et pour d’autres via une api retournant des données au format JSON. Le
backend est dockerisé.
• d’une application web construite en javascript, consommant les données JSON proposées par
l’api, et permettant de consulter et naviguer dans les données,
• d’une application mobile construite en flutter, consommant les données JSON proposées par
l’api, et permettant de consulter et naviguer dans les données.
Cette SAE permet de mettre en application des connaissances et des techniques issues de la
virtualisation, de l’architecture, de la gestion de données, de la programmation web, de la
programmation mobile.
Thème de travail : La Chaudière Agenda
La Chaudière est un lieu associatif proposant des événements culturels ouverts au grand public, dans
différentes catégories : concerts, spectacles, expositions, conférences, performances. La Chaudière
Agenda est une application en ligne permettant de consulter l’agenda des événements organisés à La
Chaudière. Ces données sont disponibles au travers d’une api, et sont utilisées par une application
web et par une application mobile permettant de faire des recherches et de consulter.
Le projet LaChaudièreAgenda est composé de trois applications :
LaChaudièreAgenda.core
LaChaudièreAgenda.core est une application backend de gestion des données de l’agenda : elle
propose une interface HTML pour créer et gérer des événements organisés à La Chaudière en les
catégorisant, et fournit une API pour rechercher et consulter l’agenda de La Chaudière.
LaChaudièreAgenda.web
LaChaudièreAgenda.web est une application frontend en javascript qui permet à ses utilisateurs
d’effectuer des recherches et de parcourir l’agenda de La Chaudière. L’application interroge l’api et
met en forme les événements pour les rendre consultables dans un navigateur web.
LaChaudièreAgenda.app
LaChaudièreAgenda.app est une application mobile en flutter qui permet de rechercher et consulter
l’agenda de La Chaudière sur un dispositif mobile..
Les événements gérés par LaChaudièreAgenda sont composées d’un titre, d’une description, d’une
date, d’une date de fin (optionnelle), d’un horaire (optionnel), d’un tarif. Des images peuvent être
associées à une entrée sous la forme d’une URL. Les événements de l’agenda sont classés par
catégories.
